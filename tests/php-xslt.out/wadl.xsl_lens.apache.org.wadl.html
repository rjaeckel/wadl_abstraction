http://localhost:8080/lens/queryapi/queries

Methods
GET

Get all the queries in the query server; can be filtered with state and queryName. This will by default only return queries submitted by the user that has started the session. To get queries of all users, set the searchAllUsers parameter to false.

request
query params
sessionid unspecified type The sessionid in which queryName is working
state unspecified type If any state is passed, all the queries in that state will be returned, otherwise all queries will be returned. Possible states are {link QueryStatus.Status#values()}
queryName unspecified type If any queryName is passed, all the queries containing the queryName will be returned, otherwise all the queries will be returned
user unspecified type Returns queries submitted by this user. If set to "all", returns queries of all users. By default, returns queries of the current user.
fromDate unspecified type from date to search queries in a time range, the range is inclusive(submitTime &gt;= fromDate)
toDate unspecified type to date to search queries in a time range, the range is inclusive(toDate &gt;= submitTime)

responses
status:

request
query params
sessionid unspecified type The session in which cancel is issued

Possible states are org.apache.lens.api.query.QueryStatus.Status#values() The queries in org.apache.lens.api.query.QueryStatus.Status#FAILED, org.apache.lens.api.query.QueryStatus.Status#CLOSED, org.apache.lens.api.query.QueryStatus.Status#SUCCESSFUL cannot be cancelled
user unspecified type If any user is passed, all the queries submitted by the user will be cancelled, otherwise all the queries will be cancelled
queryName unspecified type Cancel queries matching the query name
fromDate unspecified type the from date, inclusive(submitTime&gt;=fromDate)
toDate unspecified type the to date, inclusive(toDate&gt;=submitTime)
